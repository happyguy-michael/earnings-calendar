# GlowPing - Attention-Drawing Pulse Indicator

**Date:** 2026-03-20
**Component:** `GlowPing.tsx`
**Category:** Micro-interaction / Notification

## Inspiration

Based on 2026 UI design trends emphasizing "purposeful motion" - the idea that animation should communicate state, not merely decorate. From Tubik Studio's 2026 trends article:

> "In 2026, motion earns its keep—not by flashing, but by guiding."

Also inspired by:
- GitHub notification badges (subtle but noticeable)
- iOS notification pings
- Material Design attention patterns

## Problem

When earnings results are announced, especially for today's reports, users need a visual cue that draws attention without being intrusive. The existing FreshBadge provides text indication, but lacks the visceral "something new happened" signal that a pulse/ping animation provides.

## Solution

A versatile attention indicator with:

### Animation Variants
- **ping**: Expanding ring that fades out (like iOS notification)
- **pulse**: Subtle scale + glow pulse (like a heartbeat)
- **breathe**: Slow, ambient glow variation (calming attention)
- **ripple**: Fast expanding wave (urgent attention)
- **flash**: Quick opacity flash (strobe effect)

### Position Variants
- **dot**: Small circular indicator (corner badge style)
- **border**: Glowing border around container
- **underline**: Underline glow effect
- **full**: Full background glow
- **corner-\***: Positioned in specific corners

### Convenience Wrappers
- `NewResultPing`: For fresh earnings results (auto-colors based on beat/miss)
- `FreshDataPing`: For data freshness indication
- `ImminentPing`: For imminent events with urgency levels
- `PingDot`: Standalone dot indicator

## Integration

Added to earnings cards:
1. `NewResultPing` wraps result badges for today's reported earnings
2. `ImminentPing` wraps countdown timer for imminent reports

## Technical Details

- Respects `prefers-reduced-motion` for accessibility
- GPU-accelerated transforms (will-change, transform)
- CSS custom properties for theming
- Animation iteration counting with `onComplete` callback
- JSX-in-component styling pattern (consistent with codebase)

## Files Changed

- `src/components/GlowPing.tsx` (new)
- `src/app/page.tsx` (integration)

## Visual Impact

The ping animation creates a "something just happened" feeling that:
- Draws eye to fresh results without being annoying
- Adds urgency to imminent earnings (countdown)
- Reinforces the real-time nature of the data
